# ai-instractions
AI指示書カタログ

## 概要
ここにあるものは生成AI向けの指示書（instractions）をまとめたものです。Gemini GemやGrok Projectなどでの利用を想定しています。

それぞれ指示書の概要は以下のとおり

### Einglish
- English-Expressions.md: 英単語・英熟語を指定し例文付きで学習します
- English-Example-Texts.md: 英語の例文・和訳・IPAをテーマ別に生成します
- Anki-File-Generator.md: 入力された語彙をAnki用のインポートファイルに変換します

### Programming
- Spring-Building-Adviser.md: Spring環境に特化したアドバイスが貰えます
